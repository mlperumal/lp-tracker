🔥 One-Time Firebase Setup

Your journal will sync across Android, Windows laptop, and iPad in real-time. This takes 5 minutes once.

STEP 1

Go to console.firebase.google.com → Click Add project → Name it leaderlog → Disable Analytics → Create project

STEP 2

In your project: click the Web icon </> → Register app as leaderlog → Copy the firebaseConfig object shown

STEP 3

Go to Firestore Database → Create database → Choose production mode → Select asia-south1 (Mumbai) region

STEP 4

Go to Authentication → Get started → Enable Google sign-in. Then go to Authentication → Settings → Authorized domains and click Add domain → add your GitHub Pages URL exactly as: YOUR-USERNAME.github.io (no https, no /leaderlog). Then in Firestore → Rules, paste:
rules_version='2'; service cloud.firestore { match /databases/{db}/documents { match /users/{uid}/{doc=**} { allow read,write: if request.auth.uid == uid; } } }
